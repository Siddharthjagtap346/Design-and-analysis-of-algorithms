## **Theory**
- Maximizes **value in knapsack** with fractional items allowed  
- Uses **greedy strategy**:  
  - Sort items by **value/weight ratio**  
  - Pick full item if it fits, else take **fraction**

---

## **Code Overview**
- Input: value and weight of n items, knapsack capacity  
- Sorts items by **value/weight**  
- Calculates **maximum possible value**
